# 🐍 Classes, Objects, and Methods

Encapsulating data and functionalities within classes can create a structured and efficient approach to coding. This repository explores the fundamentals of object-oriented programming (OOP) in Python — through defining classes, instantiating them, and using their methods to perform data analyses and other data manipulations, resulting in reusability and abstraction in programming.

---

## 📚 Topics Covered

- **OOP and Python** — Introduction to object-oriented programming concepts and how Python implements them
- **Core Principles** — Understanding the foundational principles that underpin OOP design
- **Classes** — Defining and working with Python classes, including examples and hands-on exercises

## 🎯 Learning Outcomes

By the end of this module, I developed the ability to:

- Define and instantiate classes in Python to model real-world entities
- Understand and apply the core concepts of object-oriented programming (OOP) in class design
- Utilise class methods to manipulate and access the data encapsulated within objects
- Recognise the benefits of reusability and abstraction provided by classes in programming
