# 🐍 Pandas for Data Science

Given the frequent use of DataFrames in Data Science projects, this lesson delves deeper into using Pandas for manipulating and utilising DataFrame objects in Python. We explore a combination of techniques to wrangle and process data — learning how to effectively sort, filter, and transform data, as well as how to perform crucial operations like creating and deleting columns in a DataFrame.

By the end of this lesson, a solid foundation in leveraging Pandas for commonly performed data tasks is established.

---

## 📚 Topics Covered

- **Introduction to Pandas** — Understanding how to import the Pandas package, create DataFrames, and load and access the data stored within them
- **Working with DataFrames** — Manipulating DataFrames through sorting, grouping, filtering, applying transformations, and creating and deleting columns

## 🎯 Learning Outcomes

By the end of this module, I developed the ability to:

- Understand how to import the Pandas package in Python and how to create its fundamental data structure, the DataFrame
- Load and access the data stored in a DataFrame
- Work with DataFrames through practical guided exercises
- Manipulate DataFrames in various ways such as sorting, grouping, filtering, and applying transformations
