# 🐍 Data Handling in Python

NumPy is a Python package that allows us to work with data and perform operations like the loading, analysing, and storing of data. It provides high-performance, multidimensional array objects, numerical computing tools, and is fundamental in scientific computing. It also forms the basis of the Pandas library.

In this lesson, we learn how to create, manipulate, and analyse data stored in NumPy arrays.

---

## 📚 Topics Covered

- **Introduction to NumPy** — Understanding the NumPy library, its core functionality, and how it enables efficient data handling through examples and hands-on exercises

## 🎯 Learning Outcomes

By the end of this module, I developed the ability to:

- Understand the NumPy Python library and its basic functionality
- Know how to load, manipulate, and analyse data using NumPy
